# Принципы проектирования – Модуль 02

## Описание
Данный репозиторий демонстрирует корректное применение базовых принципов проектирования программного обеспечения.

В проекте рассмотрены следующие принципы:
- DRY (Don't Repeat Yourself)
- KISS (Keep It Simple)
- YAGNI (You Aren’t Gonna Need It)

## Технологии
- C#
- .NET
